## Conventions

The base URL to send all API requests is `https://api-gateway.europe.es-dis.net/v1/`. HTTPS is required for all API requests.

The Embark API follows RESTful conventions when possible, with most operations performed via `GET` or `POST `requests on page and database resources. Request and response bodies are encoded as JSON.

## Request Example

In order

<Danger>
  We actually don't know if changing the user-agent/headers or spamming the requests will result in a ban, we never tested it so please be careful!
</Danger>

```shellscript
curl -X GET 'https://api-gateway.europe.es-dis.net/v1/shared/profile' \
  -H 'Authorization: Bearer {{AUTH_TOKEN}}' 
  -H 'User-Agent: Discovery/discovery_10.00.x-CL1169805 (http-legacy) Windows/10.0.19045.1.0.64bit' 
  -H 'x-embark-request-id: {{RANDOM_ID}}'
  -H 'x-embark-manifest-id: "{{MANIFEST_ID}}"'
  -H 'x-embark-telemetry-client-platform: "3"'
  -H 'Accept: "*/*"',
  -H 'Accept-Encoding: "gzip"'
  -H 'Content-Type: application/json'
```
